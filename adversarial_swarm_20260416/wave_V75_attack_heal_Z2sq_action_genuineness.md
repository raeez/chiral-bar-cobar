(i) the idempotents $\Pi_{\pm\pm}$ to be *defined on the cochain level*
(not just cohomology);
(ii) each subspace $\Pi_{\pm\pm} \cdot \operatorname{ChirHoch}^\bullet$

projective);

- **A5**: The chain-level $(\mathbb{Z}/2)^2$-decomposition holds for
  Class A and Class B0 (automatic $E_2$ collapse) and is

**What survives**: the V68 master identity is correct for Class A and

$(\mathbb{Z}/2)^2$-action is universal for all CY_d ($d\geq 2$) admitting
