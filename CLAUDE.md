### Grading and Conventions
- COHOMOLOGICAL grading: |d| = +1 (Loday-Vallette uses homological)
- Bar uses DESUSPENSION: B(A) = T^c(s^{-1}A-bar, d) in cohomological convention
- Suspension: sV = V[-1] under V[n]^k = V^{k+n}
- Terminology: "strict" not "on-nose" for d^2=0

### Koszul Duality
- Com^! = Lie (NOT coLie). Sym^! = Lambda (exterior).
- Koszul dual coalgebra: SUB-coalgebra of cofree (cogenerated by R-perp), NOT quotient
- Heisenberg NOT self-dual: H^! = Sym^ch(V*), NOT H_{-k}
- Free fermion: F^! = beta-gamma (Lie<->Com duality), NOT Heisenberg
- bc-betagamma is a 2-generator duality (dim V=2). Bosonization != Koszul duality.
- CHIRAL KOSZULNESS != CLASSICAL: U(g) Koszul (BGS) does NOT imply g-hat_k is chiral Koszul. Chiral bar uses all OPE poles (Borcherds). Separate proof needed.
- Bar differential: d_bracket^2 != 0 (PROVED, all 2048 signs). Full d = d_bracket + d_curvature satisfies d^2=0 via Borcherds. Do NOT build d_bracket as a matrix.
- Correct method: PBW spectral sequence + Koszul dual Hilbert series. NOT matrix rank.
- PBW SS does NOT automatically degenerate at E_3 for "quadratic OPE" — composites (e.g. Sugawara) have higher poles from double Wick contractions.

### P-infinity-Chiral vs Coisson
- **Coisson** (BD) = PVA = commutative D_X-algebra + lambda-bracket. NOT a chiral algebra.
- **P-infinity-chiral** = chirCom + chirLie with Leibniz. IS a chiral algebra (E-infinity + L-infinity).
- Different quantization levels: Coisson -> E-infinity-chiral (singly quantum), P-infinity-chiral -> E_1-chiral (doubly quantum).

### Central Charges and Levels
- Sugawara: T = (1/2(k+h-dual))Sum :J^a J^a:, c = k*dim(g)/(k+h-dual)
- Sugawara UNDEFINED at critical level k=-h-dual (not "c diverges")

- Virasoro DS: c = 1 - 6(k+1)^2/(k+2); W_3 DS: c = 2 - 24(k+2)^2/(k+3)
